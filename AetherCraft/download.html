手机端安装指南

1. 准备

首先需要一个脑子，一只手，一部手机，ZeroTermux/Termux软件(百度搜)

2. 基本环境配置

安装完成后，打开ZeroTermux并粘贴以下命令回车以更新系统(国内)：

bash -c '
# 定义所有可用的国内镜像组
MIRRORS=(
    "https://mirrors.hust.edu.cn/termux/apt/termux-main"
    "https://mirrors.nju.edu.cn/termux/apt/termux-main"
    "https://mirrors.cernet.edu.cn/termux/apt/termux-main"
    "https://mirror.nyist.edu.cn/termux/apt/termux-main"
    "https://mirrors.sau.edu.cn/termux/apt/termux-main"
    "https://mirrors.cqupt.edu.cn/termux/termux-main"
    "https://mirror.iscas.ac.cn/termux/apt/termux-main"
    "https://mirrors.pku.edu.cn/termux/termux-main/"
    "https://mirrors.tuna.tsinghua.edu.cn/termux/apt/termux-main"
    "https://mirrors.bfsu.edu.cn/termux/apt/termux-main"
    "https://mirrors.zju.edu.cn/termux/apt/termux-main"
    "https://mirrors.sustech.edu.cn/termux/apt/termux-main"
    "https://mirror.sjtu.edu.cn/termux/termux-main/"
    "https://mirrors.aliyun.com/termux/termux-main"
    "https://mirrors.ustc.edu.cn/termux/termux-main"
)

# 备份原来的sources.list文件
if [ -f "$PREFIX/etc/apt/sources.list" ]; then
    cp "$PREFIX/etc/apt/sources.list" "$PREFIX/etc/apt/sources.list.bak"
fi

# 生成新的sources.list文件
echo "# Termux 国内镜像组配置" > "$PREFIX/etc/apt/sources.list"
echo "# 生成时间: $(date)" >> "$PREFIX/etc/apt/sources.list"
echo "" >> "$PREFIX/etc/apt/sources.list"

# 添加主仓库(main)镜像
echo "# 主仓库镜像" >> "$PREFIX/etc/apt/sources.list"
for mirror in "${MIRRORS[@]}"; do
    BASE_URL=${mirror%/*}
    echo "deb ${BASE_URL}/ stable main" >> "$PREFIX/etc/apt/sources.list"
done

# 添加游戏仓库(games)镜像
echo "" >> "$PREFIX/etc/apt/sources.list"
echo "# 游戏仓库镜像" >> "$PREFIX/etc/apt/sources.list"
for mirror in "${MIRRORS[@]}"; do
    BASE_URL=${mirror%/*}
    echo "deb ${BASE_URL}/ games stable" >> "$PREFIX/etc/apt/sources.list"
done

echo "Termux 软件源已成功更换为国内镜像组，包含以下镜像:"
for mirror in "${MIRRORS[@]}"; do
    BASE_URL=${mirror%/*}
    echo "  - ${BASE_URL}"
done
echo ""
pkg up
echo "更换成功"
'

3. 运行Aether_Craft脚本

使用curl执行：

pkg install -y curl ; bash -c "$(curl -L https://gitee.com/jiangnan-qwq/AetherCraft_cn/raw/main/main.sh)"
pkg install -y curl ; bash -c "$(curl -L https://raw.githubusercontent.com/JiangNan-QwQ/AetherCraft/main/main.sh)"

注意选择速度快的站。

电脑端安装指南

1. 系统要求

Aether_Craft支持以下操作系统：

  • Linux发行版（Ubuntu/Debian及基于Debian的系统）

建议系统配置：

  • 至少2GB内存
  • 10GB可用磁盘空间
  • Java 21

2. 安装依赖

根据您的所在地，换源(国外跳过)，如没有您的系统，请自行寻找换源教程(限apt包管理器)：

#无代码替换方案，请上网自行搜索
sudo sed -i "s@http://http.kali.org/kali@https://mirrors.tuna.tsinghua.edu.cn/kali@g" /etc/apt/sources.list

3. 运行Aether_Craft脚本

使用curl执行：

apt install -y curl ; bash -c "$(curl -L https://gitee.com/jiangnan-qwq/AetherCraft_cn/raw/main/main.sh)"
apt install -y curl ; bash -c "$(curl -L https://raw.githubusercontent.com/JiangNan-QwQ/AetherCraft/main/main.sh)"

注意选择速度快的站。

常见问题解答

怎么管理服务器？

这里讲的是运行界面的管理，更多管理功能请见脚本主界面。

停止服务器：

stop

查看日志：

tail -f logs/latest.log

如何更新Aether_Craft到最新版本？

无需更新，每次运行都会获取最新版本！

服务器启动后无法连接怎么办？

请检查以下事项：

  1. 确保服务器已完全启动（等待"Done"消息）
  2. 检查防火墙设置是否允许服务器端口
  3. 确认server.properties中的online-mode设置
  4. 如果是局域网连接，确保设备在同一网络，并确认连接的IP地址
  5. 如果是外部连接，请配置好内网穿透

您还可以查看日志文件获取更多信息：

cat logs/latest.log | grep -i error

如何备份我的服务器？

Aether_Craft提供了备份功能，详见主界面。
